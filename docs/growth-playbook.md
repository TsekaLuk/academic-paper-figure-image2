# Growth Playbook

This skill can grow because every reviewer complaint becomes a reusable rule.

## Growth Loop

```text
rejected page
  → classify complaint
  → add audit rule
  → update prompt template
  → produce better figure
  → document before/after
  → reuse on next thesis
```

## Acquisition Hook

The message is simple:

> "Your system may be enough. The figures and placement are what get you rejected."

This targets users who already have a thesis draft but are blocked by conservative formatting and diagram expectations.

## Activation

Give the skill:

- thesis PDF/Word/LaTeX;
- accepted sample theses;
- advisor comments;
- source code or screenshots;
- current figure list.

The first deliverable should be a visible risk map, not rewritten images.

## Retention

Each solved thesis adds:

- new chapter-placement examples;
- new diagram prompt patterns;
- new rejection causes;
- stronger before/after assets;
- more precise DO/DO NOT rules.

## Referral Asset

The strongest sharable artifact is a side-by-side comparison:

- before: Mermaid/TikZ/mock/missing figure;
- after: traditional academic figure placed correctly in the chapter;
- note: reviewer complaint resolved.

Do not market this as "AI art". Market it as "thesis figure risk reduction".
